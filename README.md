- Designed, engineered, and shipped **iOS + Android** solo — from first commit to public **App Store & Google Play** launch
- Architected a **TypeScript / React Native + Expo** codebase across 20 screens, an Anchor panic-attack flow, and an iOS home-screen widget
- Built a **regression-grade QA harness** (Jest unit tests + Maestro E2E smoke & regression) wired into **GitHub Actions CI** for daily releases
- Stood up the full production stack: **PostHog, Sentry, Supabase, RevenueCat** (3 live tiers), Apple/Google Sign-In, EAS release automation
- Shipped **i18n for 6 locales** (EN, DE, ES, FR, IT, HR) on i18next
